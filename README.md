- 实践是检验真理的唯一标准

我是一个擅长计算机网络的技术从业人员，如果你问我网络技术如何掌握，我认为在问这个问题前，先理解下互联网的根基**TCP/IP**

- 答：“防火墙”,主要涉及以下协议和技术

  - DNS
  - TLS
  - TCP/UDP
  - IP(VLSM/CIDR)
  - VPN
  - WireShark
  - IP Routing
    - Default Routing
    - Direct Routing
    - Static Routing
  - Packet Filter
  - NAT（DNAT/SNAT）
  - 基于策略的路由转发
